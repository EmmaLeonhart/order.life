# Other mythic cycles — weaker-tier but usable

Cultures in `structure.md` §6's "Other candidates (weaker but usable)"
list. Each gets a brief note here; if any firms up into a dedicated
chapter it should get split out into its own file per the directory
convention.

## Tibet

- **Yarlung dynasty** traces to **Nyatri Tsenpo** (sky-descended).
- Documented in *Red Annals* (14th c.), *Blue Annals* (15th c.).
- Bridge is reasonably clean; register is Buddhist-filtered by the time
  of compilation (structurally similar to the Confucian filter in
  `china.md`).

## Korea

- *Samguk Yusa* (1281). **Dangun** founding myth plus **Park
  Hyeokgeose**, **Kim Alji** bridging to historical
  Silla / Goguryeo / Baekje.
- Connects to `heo-hwang-ok.md` — Ayodhya → Korea → Japan is the one
  named Asian-to-Asian bridge figure in the gap, and she lands in the
  Silla-era material.

## Vietnam

- Hồng Bàng dynasty via **Lạc Long Quân** and **Âu Cơ**. *Lĩnh Nam
  chích quái* (14th c.), *Đại Việt sử ký toàn thư* (15th c.).
- Standard mythic-to-historical chain; the documentation is
  post-Chinese-influence so somewhat rationalized (see `china.md`).

## Wales

- Mabinogion + Geoffrey of Monmouth's *Historia Regum Britanniae*
  (1136). Arthur → Uther → Constantine → back through Roman Britain →
  Brutus of Troy.
- Vehicle for the British-Arthur fix referenced in `structure.md` §7
  (Gaiad has a specific move to make here).
- Plays well with `rome.md` via the Brutus-of-Troy inheritance.
