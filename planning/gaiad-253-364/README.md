# Modern Period Planning — `gaiad-253-364`

Working planning folder for the Modern Period of the Gaiad (ch 253–362
human-scale content; ch 363/364 structurally separate post-human
finale).

## This is a living folder

The Modern Period is explicitly planned to be **updated regularly**,
not settled once and frozen. Reasons:

- The book's last depicted chapter is ch 362 (2026 → time skip). That's
  the present. As 2026 keeps happening, the content of the closing
  arc may need to shift.
- The three thematic axes (invented tradition, suppressed syncretism,
  technological transformation) are still developing in real time —
  especially AI, climate, and reactionary movements.
- We are intentionally **expanding** the planning over time rather
  than crash-drafting a fixed plan. Per the project convention,
  drafting is gated on Leo (2026-08-12 onward). Until then, planning
  is the work.

Treat every file here as a working document. Nothing here is
permanent except the hard structural anchors.

- **Ch 253** — Chronicle of the Popes (Peter → Alexander VI)

if the thematic shape demands it.
